## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js 18.17 or later
- npm, yarn, pnpm, or bun

<Callout type="tip" title="Recommended">
  We recommend using pnpm for faster installation and better disk space usage.
</Callout>

## Create New Project

The easiest way to get started with nene.js is using our CLI to create a new project:

```bash
npx create-nene@latest my-app
cd my-app
npm run dev
```

This will:

1. Create a new directory called `my-app`
2. Set up a complete nene.js project structure
3. Install all dependencies
4. Generate example controllers and hooks

## Manual Installation

If you prefer to set up nene.js manually in an existing project:

<Tabs defaultValue="npm">
  <TabList>
    <TabTrigger value="npm">npm</TabTrigger>
    <TabTrigger value="pnpm">pnpm</TabTrigger>

  </TabList>

{" "}
<TabContent value="npm">```bash npm install nene ```</TabContent>

{" "}
<TabContent value="pnpm">```bash pnpm add nene ```</TabContent>

  <TabContent value="yarn">
    ```bash
    yarn add nene
    ```
  </TabContent>
</Tabs>

Then update your `package.json` scripts:

```json
{
  "scripts": {
    "dev": "nene dev",
    "build": "nene build",
    "start": "nene start"
  }
}
```

Create the required directories:

```bash
mkdir -p src/controllers src/services src/hooks
```
