## 事前要件

始める前に、以下がインストールされていることを確認してください：

- Node.js 18.17 以上
- npm、yarn、pnpm、または bun

<Callout type="tip" title="推奨事項">
  より高速なインストールと効率的なディスク使用のために、pnpmの使用を推奨します。
</Callout>

## 新しいプロジェクトの作成

nene.js を始める最も簡単な方法は、CLI を使用して新しいプロジェクトを作成することです：

```bash
npx create-nene@latest my-app
cd my-app
npm run dev
```

このコマンドは以下を実行します：

1. `my-app`という新しいディレクトリを作成
2. 完全な nene.js プロジェクト構造を設定
3. すべての依存関係をインストール
4. サンプルコントローラーとフックを生成

## 手動インストール

既存のプロジェクトに nene.js を手動で設定するには：

<Tabs defaultValue="npm">
  <TabList>
    <TabTrigger value="npm">npm</TabTrigger>
    <TabTrigger value="pnpm">pnpm</TabTrigger>
    <TabTrigger value="yarn">yarn</TabTrigger>
  </TabList>

{" "}
<TabContent value="npm">```bash npm install nene ```</TabContent>

{" "}
<TabContent value="pnpm">```bash pnpm add nene ```</TabContent>

  <TabContent value="yarn">
    ```bash
    yarn add nene
    ```
  </TabContent>
</Tabs>

次に、`package.json`のスクリプトを更新してください：

```json
{
  "scripts": {
    "dev": "nene dev",
    "build": "nene build",
    "start": "nene start"
  }
}
```

必要なディレクトリを作成してください：

```bash
mkdir -p src/controllers src/services src/hooks
```
