## 사전 요구사항

시작하기 전에 다음이 설치되어 있는지 확인하세요:

- Node.js 18.17 이상
- npm, yarn, pnpm, 또는 bun

<Callout type="tip" title="권장사항">
  더 빠른 설치와 효율적인 디스크 사용을 위해 pnpm 사용을 권장합니다.
</Callout>

## 새 프로젝트 생성

nene.js를 시작하는 가장 쉬운 방법은 CLI를 사용하여 새 프로젝트를 생성하는 것입니다:

```bash
npx create-nene@latest my-app
cd my-app
npm run dev
```

이 명령은 다음을 수행합니다:

1. `my-app`이라는 새 디렉토리 생성
2. 완전한 nene.js 프로젝트 구조 설정
3. 모든 의존성 설치
4. 예제 컨트롤러와 훅 생성

## 수동 설치

기존 프로젝트에 nene.js를 수동으로 설정하려면:

  <TabList>
    <TabTrigger value="npm">npm</TabTrigger>
    <TabTrigger value="pnpm">pnpm</TabTrigger>
    <TabTrigger value="yarn">yarn</TabTrigger>

<TabContent value="npm">```bash npm install nene ```</TabContent>

<TabContent value="pnpm">```bash pnpm add nene ```</TabContent>

  <TabContent value="yarn">
    ```bash
    yarn add nene
    ```
  </TabContent>
</Tabs>

그런 다음 `package.json` 스크립트를 업데이트하세요:

```json
{
  "scripts": {
    "dev": "nene dev",
    "build": "nene build",
    "start": "nene start"
  }
}
```

필요한 디렉토리를 생성하세요:

```bash
mkdir -p src/controllers src/services src/hooks
```
