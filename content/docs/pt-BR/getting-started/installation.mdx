## Pré-requisitos

Antes de começar, certifique-se de ter os seguintes itens instalados:

- Node.js 18.17 ou superior
- npm, yarn, pnpm ou bun

<Callout type="tip" title="Recomendado">
  Recomendamos usar o pnpm para instalação mais rápida e melhor uso de espaço em
  disco.
</Callout>

## Criar Novo Projeto

A maneira mais fácil de começar com o nene.js é usar nossa CLI para criar um novo projeto:

```bash
npx create-nene@latest my-app
cd my-app
npm run dev
```

Isso irá:

1. Criar um novo diretório chamado `my-app`
2. Configurar uma estrutura completa de projeto nene.js
3. Instalar todas as dependências
4. Gerar controllers e hooks de exemplo

## Instalação Manual

Se você preferir configurar o nene.js manualmente em um projeto existente:

  <TabList>

{" "}
<TabContent value="npm">
  ```bash npm install nene @nene/core @nene/cli ```
</TabContent>

{" "}
<TabContent value="pnpm">
  ```bash pnpm add nene @nene/core @nene/cli ```
</TabContent>

  <TabContent value="yarn">
    ```bash yarn add nene @nene/core @nene/cli ```
  </TabContent>
</Tabs>

Em seguida, adicione a configuração do nene.js ao seu projeto:

```typescript
// nene.config.ts
import { defineConfig } from "@nene/core";

export default defineConfig({
  controllersDir: "src/controllers",
  hooksOutputDir: "src/hooks",
  apiPrefix: "/api",
});
```
