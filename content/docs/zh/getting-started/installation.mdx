## 先决条件

在开始之前，请确保已安装以下内容：

- Node.js 18.17 或更高版本
- npm、yarn、pnpm 或 bun

<Callout type="tip" title="推荐">
  建议使用 pnpm 以获得更快的安装速度和更高效的磁盘使用。
</Callout>

## 创建新项目

开始使用 nene.js 最简单的方法是使用 CLI 创建新项目：

```bash
npx create-nene@latest my-app
cd my-app
npm run dev
```

此命令将执行以下操作：

1. 创建一个名为 `my-app` 的新目录
2. 设置完整的 nene.js 项目结构
3. 安装所有依赖项
4. 生成示例控制器和钩子

  <TabContent value="yarn">
    ```bash yarn add nene @nene/core @nene/cli ```
  </TabContent>
</Tabs>

然后在项目中添加 nene.js 配置：

```typescript
// nene.config.ts
import { defineConfig } from "@nene/core";

export default defineConfig({
  controllersDir: "src/controllers",
  hooksOutputDir: "src/hooks",
  apiPrefix: "/api",
});
```
